# Gestión de Tienda con Códigos QR - Alejandro Tovar

Este proyecto es un sistema de gestión de inventario para una tienda de calzado desarrollado con **Spring Boot 3**.

## 🛠 Tecnologías utilizadas
* **Java 17**
* **Spring Boot 3.2.0**
* **Spring Data JPA** (Persistencia de datos)
* **Spring Security** (Cifrado BCrypt para contraseñas)
* **Lombok** (Productividad en el código)
* **MySQL** (Base de datos mediante XAMPP)

##  Estado del Proyecto (Hito 1 - 25%)
- [x] Estructura de paquetes profesional (Entity, Repository, Service, Controller).
- [x] Modelo de base de datos implementado (Usuarios, Productos, Categorías).
- [x] Sistema de seguridad con BCrypt implementado en el Service.
- [x] Conexión exitosa con la base de datos MySQL.

##  Requisitos para ejecución
1. Tener instalado **XAMPP**.
2. Importar el archivo `SQLTFG.sql` en phpMyAdmin.
3. Ejecutar la aplicación desde el archivo `App.java`.
